# Custom Discord Bot Developer

I build **fully custom Discord bots** tailored to whatever your server actually needs — not templates, not one-size-fits-all. Tell me what you want and I'll build it from scratch.

---

## What I Can Build

**Moderation systems** — ban/kick/timeout/warn commands, role-gated so only staff can use them, automatic DMs to punished users with reasons and appeal links, full mod-action logging, hierarchy protection so staff can't punish higher-ranked members.

**Server logging** — track message edits/deletes, member joins/leaves, role and channel changes, voice activity, bans/unbans with audit-log reasons — all posted cleanly to a log channel.

**Verification systems** — integrate with tools like Bloxlink, custom nickname formatting, verified-only channel access, verification status checks.

**Ticket / support systems** — private ticket channels, auto-welcome messages, staff claiming, escalation flows.

**Helper / support-staff tools** — FAQ commands, guides, private staff notes on members, activity tracking for support teams.

**Game-specific integrations** — Roblox profile lookups, external API pulls, custom embeds showing whatever data your community cares about.

**Custom features** — if it's not listed above, ask anyway. Economy systems, leveling, giveaways, polls, reaction roles, welcome messages, auto-moderation filters, scheduled announcements — if it can run on Discord, I can likely build it.

---

## How I Work

Every bot is built around **your** server's actual setup — your roles, your channels, your rules — not a generic template you have to bend your server to fit. I write clean, modular code (commands and features split into organized files) so it's easy to extend later if you want more added.

**Tech stack:** Node.js, discord.js v14, built to deploy on any Node-friendly host (Railway, Render, etc.).

**Note:** I write and deliver the bot's code — hosting is not included. You'll need your own hosting service (I can point you toward beginner-friendly options and help you get set up, but running/paying for the server is on your end).
